nbc command line tools
----------------------

&nbsp;

### Install nbcc

``` bash
git clone https://github.com/fn-share/nbcc.git
cd nbcc
python3 setup.py install
```

or,

``` bash
pip3 install git+https://github.com/fn-share/nbcc.git
```

&nbsp;

### Command Line

``` bash
$ nbcc --help
Usage: nbcc [OPTIONS] COMMAND [ARGS]...

Options:
  --help  Show this message and exit.

Commands:
  account
  app
  autostart
  deploy
  init
  localaccess
  server
```
